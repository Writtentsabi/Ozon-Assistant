# OxyZen Browser Assistant 🧘‍♂️✨

Ο **OxyZen Browser Assistant** είναι ένας προηγμένος, multimodal βοηθός τεχνητής νοημοσύνης (AI) ενσωματωμένος στο οικοσύστημα του OxyZen Browser. Χρησιμοποιεί το μοντέλο **Gemini 2.0 Flash** για να παρέχει έξυπνες απαντήσεις, ανάλυση εικόνων και μια μοναδική λειτουργία "εσωτερικής σκέψης" (Chain of Thought).

## 🚀 Δυνατότητες

- **Multimodal Chat**: Δυνατότητα αποστολής κειμένου και εικόνων ταυτόχρονα για ανάλυση.
- **Internal Monologue (Chain of Thought)**: Ο Zen εμφανίζει τις σκέψεις του πριν την τελική απάντηση σε ειδικό block, προσφέροντας διαφάνεια στον τρόπο σκέψης της AI.
- **Πολυγλωσσική Υποστήριξη**: Πλήρως λειτουργικό σε Ελληνικά και Αγγλικά με αυτόματη εναλλαγή.
- **Material Design UI**: Μοντέρνο περιβάλλον εργασίας με υποστήριξη Dark Mode και responsive σχεδιασμό για κινητά και desktop.
- **Stateless History**: Διατήρηση ιστορικού συνομιλίας στον client για συνεχή διάλογο.

## 🛠️ Τεχνολογίες

- **Frontend**: HTML5, CSS3 (Material UI), JavaScript (ESModules).
- **Backend**: Node.js, Express.
- **AI Engine**: Google Generative AI (Gemini API).
- **Icons**: Font Awesome.
